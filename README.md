# Loa_Wandering_merchant

![로아 떠상 알림봇](/image/wei.png)

로아 떠돌이 상인 물품 알림봇

로스트아크 api에서 떠돌이상인을 제공하고 있지 않습니다.
KLOA 의 떠돌이상인 https://kloa.gg/merchant 페이지를 통해 알림을 디스코드로 전송합니다.
(이는 KLOA 사이트에 온전히 의존하고 있습니다.)
